# jump-model-verification

Computational verification of the long jump and high jump model from:

O. Helene and M. T. Yamashita, "A unified model for the long and high jump," American Journal of Physics 73, 906-908 (2005).

## Files

- `jump_model_verification.ipynb`: Google Colab notebook containing the Python code and numerical results
- `long_jump_energy_vs_distance.png`: Graph of predicted long jump distance as a function of additional energy
- `high_jump_angle_vs_height.png`: Graph of predicted high jump height as a function of body inclination angle
- `jump_model_results.csv`: Summary table comparing Python results with paper values

## Main Results

The Python calculation reproduces the main numerical estimates in the original paper, including:

- Additional energy: 550 J
- Long jump distance: about 8.37 m
- Long jump takeoff angle: about 20.35 degrees
- High jump final vertical velocity: about 4.53 m/s
- High jump total height: about 2.15 m
- High jump takeoff angle: about 49.54 degrees

## Purpose

This repository was created for the computational verification section of a General Physics I mechanics paper study project.
